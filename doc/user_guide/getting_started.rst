Getting Started
===============

Browse the various code snippets available in the **/neurospin/gaia/neurospin-cook-book** shared folder and copy/paste some code snippets into your own project.

To contribute, simply add/edit files in the **unit_examples** and **integrated_examples** folders.
The **unit_examples** folder is intended to contain utility code that can be reused in different contexts.
The **integrated_examples** folder is intended to contain more self-contained code that can somehow be assimilated into tutorials.

The synthax must follow the `Sphynx Gallery <https://sphinx-gallery.github.io/stable/getting_started.html>`_ structure. Within the **unit_examples** and **integrated_examples** folders, examples are organized by category.
Feel free to add new categories and don't forget to create a `README.rst` file in each new folder.

If you want to share Jupyter notebooks, please first convert them with the following command: :code:`jupyter nbconvert --to script *.ipynb`.

Please note that you need to apply an appropriate license in your project (usually a `CeCILL-B license <http://www.cecill.info>`_).
